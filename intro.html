Welcome to Use Learning Enabled Optimization Software

The guidance for LEO method is shown below:

You need to run from Input Data to Validation part by yourself first, and generate the validation cost-to-go data.

This software can help you do the test part and generate the test data in log_data_time.txt file.

Software guidance:

1. In File menu, generate a log_data_time.txt file which will contains all the output information from this software

2. In Model Validation menu, first choose the the cost-to-go output files for training data and validation data generated by one model. Then do T-test, F-test, and χ2-test.

3. In Compare menu, first choose all the cost-to-go output files for validation data generated by all models. Then do the Kruskal-Wallis Test, and draw the CDF figure to compare different model.
